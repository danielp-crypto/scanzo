How SCAN-ZO works

A simple system to help schools return lost clothes to the right family — fast.

1
Parents register each item — name, class, description, and WhatsApp number.
2
A QR code is generated and printed. Attach it to the clothing with a label or sticker.
3
When the item is found, anyone scans the QR — no app, no login needed.
4
One tap sends a WhatsApp message to the parent — item returned, problem solved.

Powered by a shared school database

All registered items are stored in a live Firebase database. Any teacher or parent on any device can look up an item by ID — no account needed.
